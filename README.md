## Task 1: Understanding Dataset & Data Types

This repository contains the work for Task 1 of the AI & ML Internship at the Elevate Labs.

### Objective
To understand the structure of a dataset, identify data types, analyze missing values, and evaluate suitability for machine learning.

### Work Done
- Loaded the Titanic dataset
- Viewed first and last records
- Analyzed dataset size
- Identified numerical, categorical, ordinal, and binary features
- Checked missing values and data imbalance
- Identified target variable for ML

### Tools Used
- Python
- Pandas
- NumPy
- Google Colab
